# Week 1: REST API Setup

## Objective
Build a functional REST API using the MERN stack (MongoDB, Express, Node.js) following the freeCodeCamp guide.

## Technologies Used
* **Node.js** & **Express** for the server.
* **Mongoose** for database modeling.
* **MongoDB Atlas** as the cloud database provider.
* **Dotenv** for environment variable management.

## Features Implemented
- [x] Connection to MongoDB Atlas Cluster.
- [x] Environment variables configuration (`.env`).
- [x] Data Schema (Name and Age).
- [x] CRUD Endpoints (POST and GET implemented).

## How to run locally
1. `cd week-1`
2. `npm install`
3. Create a `.env` file with your `DATABASE_URL`.
4. Run `node index.js`.
